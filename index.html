How engagements work

Week 1 — discovery & spec. You give me API access + a target user workflow. I produce a 1-page spec: which endpoints to wrap, which tools to expose, what the agent should be able to do.

Week 2-3 — build & ship. Fork the template, customise the client, register tools, write tests. You get a working MCP server with documentation, ready to deploy to your Claude / Cursor users.

Week 4 — harden & handoff. Multi-tenant auth, rate-limit handling, audit logging, deployment guide. You own everything — full source, MIT license, no vendor lock-in.
